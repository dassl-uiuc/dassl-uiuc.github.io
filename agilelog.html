AgileLog

AgileLog is the substrate to build exploratory AI agents reliably. Read the research paper.

view resources GitHub

agilelog@dassl:~$ agilelog --client append log "hello agent!"

Record offset: 0

agilelog@dassl:~$ agilelog --client cfork log child1

agilelog@dassl:~$ agilelog --client append log "continuous"

Record offset: 1

agilelog@dassl:~$ agilelog --client append child1 "hello child!"

Record offset: 2

agilelog@dassl:~$ agilelog --client readall child1

agilelog@dassl:~$ "hello agent!", "continuous", hello child!"

agilelog@dassl:~$

Overview

Log-Centric Design

The log is the data primitive for agent coordination, ordering, and recovery.

Forking

Natively supports forking to allow exploratory agentic execution.

Low-Latency Operation

Focuses on reducing end-to-end delay for applications that depend on fast decisions.
